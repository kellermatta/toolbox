Instrucciones para ejecutar el program

Utilizando docker

1) Descargar docker client version 20.10.2
2) Clonar el proyecto utilizando git clone + url
3) abrir el directorio utilizando cd toolboxapps
4) correr el comando docker-compose up
5) en el navegador abrir http://localhost:3000

utilizando npm
    1) instalar nodejs 12
    2) 2) Clonar el proyecto utilizando git clone + url
    3) abrir el directorio utilizando cd toolboxapps/toolbox para el backend
    4) instalar dependencias utiizando npm install
    5) npm start o si se quiere probar los test escribir npm run test
    6) escribir cd .. para regresar un directorio
    7) abrir el directorio toolboxapps/toolbox-front
    8) instalar las dependecias npm install
    9) npm start
